<h1>Portfolio Website</h1>

<p>
Website ini merupakan portfolio pribadi yang dibuat untuk menampilkan informasi mengenai profil diri, pengalaman, serta kontak yang dapat dihubungi. Website ini dikembangkan menggunakan HTML dan CSS sebagai dasar dalam pembuatan tampilan web.
</p>

<p>
Fitur yang terdapat pada website ini meliputi halaman profil, deskripsi tentang saya, pengalaman proyek, serta form kontak sederhana. Website ini juga telah diupload ke GitHub menggunakan Git sebagai version control untuk mempermudah pengelolaan project.
</p>
