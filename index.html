Kontribusi Saya (Backend Developer):

  • Mengembangkan REST API dengan 100+ endpoint menggunakan Express.js dan PostgreSQL
  • Mengimplementasikan Autentikasi & Otorisasi berbasis JWT dengan 3 peran (pembeli, penjual, dan admin)

Struktur Database (22 Tabel Relasional):

  • pengguna - Manajemen user dengan role (pembeli/penjual/admin)
  • toko - Manajemen toko untuk penjual
  • produk - Data produk dengan stok, harga, rating
  • kategori - Kategorisasi produk
  • product_images - Multiple gambar per produk
  • keranjang - Shopping cart user
  • pesanan - Manajemen order/transaksi
  • item_pesanan - Detail item dalam pesanan
  • alamat - Alamat pengiriman user
  • voucher - Manajemen diskon dan promo
  • ulasan - Review & rating produk oleh pembeli
  • wishlist - Daftar wishlist user
  • laporan - Sistem pelaporan konten
  • notifikasi - Notifikasi real-time user
  • chat_messages - Fitur chat antar pengguna
  • follow_toko - Follow toko favorit
  • payments_komerce - Integrasi payment gateway
  • transaksi - Riwayat pembayaran
  • saldo & saldo_history - Sistem saldo/dompet digital
  • loyalty_points - Sistem poin loyalitas
  • search_history - Riwayat pencarian user
  • blogs - Manajemen konten/blog
  • settings - Pengaturan sistem

Teknologi: Node.js, Express.js, PostgreSQL, JWT, Railway, Vercel, Postman, Git
